## 👤 About

50 practical AWS labs completed on the [KodeKloud — 100 Days of Cloud](https://learn.kodekloud.com) program, covering core services from EC2 and IAM through to EKS, Lambda, and CloudFormation. Every lab was executed in a live AWS environment with real infrastructure — no sandboxes, no shortcuts. This repository documents the full journey as proof of consistent, hands-on cloud practice.

## 🧠 Key Skills Gained

- **Compute** — EC2 lifecycle management, AMIs, instance types, EBS volumes, snapshots, Auto Scaling
- **Networking** — VPC design, subnets, security groups, Elastic IPs, ENIs, NAT Gateway, NAT Instance, VPC Peering, ALB
- **Identity & Security** — IAM users, groups, roles, policies, KMS encryption
- **Storage** — S3 versioning, static website hosting, CLI migrations, ECR repositories
- **Databases** — RDS provisioning, snapshots & restore, DynamoDB NoSQL operations
- **Serverless & Events** — Lambda (console + CLI), S3 event triggers, SQS & SNS messaging
- **Containers & Orchestration** — ECS deployments, EKS Kubernetes cluster management
- **Observability** — CloudWatch alarms, VPC audit logging
- **Infrastructure as Code** — AWS CloudFormation stack deployments
